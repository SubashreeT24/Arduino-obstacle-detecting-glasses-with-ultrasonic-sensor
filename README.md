## 📖 1. Overview

A wearable assistive device built into a spectacle frame, designed to alert users about nearby obstacles in real time. It uses an ultrasonic sensor to measure distance and provides immediate buzzer/vibration feedback — supporting safer, more independent navigation, particularly for visually impaired individuals.

---

## 📌 2. Goal

To support safer navigation for **visually impaired individuals** using real-time, distance-based obstacle alerts.

---

## ⭐ 3. Key Features

- 📏 Compact design, built directly into a spectacle frame
- 🙌 Hands-free usage
- 💰 Low-cost solution
- ⚡ Real-time feedback on obstacle proximity

---

## 🛠️ 4. Components Used

| Component | Purpose |
|---|---|
| **Arduino UNO** | Processes sensor data and triggers alerts |
| **Ultrasonic Sensor (HC-SR04)** | Measures distance to nearby obstacles |
| **Buzzer / Vibration Motor** | Alerts the user when an obstacle is detected |
| **Power Bank** | Portable power supply (used for demo/testing) |
| **Spectacle Frame** | Wearable housing for all components |

---

## ⚙️ 5. How It Works

1. The **ultrasonic sensor (HC-SR04)** continuously measures the distance to objects ahead
2. The **Arduino UNO** processes the distance data in real time
3. When an obstacle is detected within a set threshold distance, the **buzzer or vibration motor** alerts the user
4. The system resets and continues monitoring as the user moves
